# OpenHacienda

**Documentación no oficial sobre la AEAT y la administración tributaria española**

OpenHacienda es un recurso open source para desarrolladores, integradores y ciudadanos que quieren entender cómo funcionan los sistemas de la Agencia Tributaria española — tanto a nivel técnico como institucional.

La documentación oficial de la AEAT es frecuentemente incompleta, dispersa o difícil de interpretar. Este proyecto intenta llenar ese hueco con dos objetivos:

1. **Documentación técnica** para quienes integran con los sistemas de la AEAT: APIs, formatos, certificados, autenticación.
2. **Documentación institucional** para quienes quieren entender cómo funciona la administración tributaria: contratación pública, adjudicatarios, cómo buscar información y cómo ejercer derechos.

:::warning[Aviso legal]

Este sitio **no está afiliado con la AEAT** ni con ningún organismo oficial. La información aquí publicada es de carácter técnico y orientativo.
:::
